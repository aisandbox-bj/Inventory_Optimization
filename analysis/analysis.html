Analysis Engine

Pipeline ports of scripts 02 / 03 / 04. Charts. LLM review. Excel export.

The analysis engine consumes the canonical JSON the intake engine produces and runs the deterministic MRP pipeline:

  • Multi-model detection (when scope is fleet with ≥ 2 models) — materials consumed across multiple fleets routed to a separate MULTI bucket.
  • Per-bucket extraction — fleet × model, manual list, classification group, or vendor.
  • Period rates — P1 baseline (5 mo) + P2 rolling (3 mo back from run date).
  • HCE detection — single WO ≥ 50 % of period total, or ≥ 3× the average WO qty.
  • Lumpy / smooth classification — coefficient of variation + top-WO share.
  • Recommended Min / Max + 8-rule traffic-light decision.
  • Per-material chart with cumulative actual + P1 / P2 trend lines + WO annotations.
  • LLM review surface — provider-agnostic, configured in Settings. Reviews chart images and proposes JSON edits the operator approves.
  • Excel export via ExcelJS — one workbook per bucket, embedded charts, action list.
