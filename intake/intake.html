Data alignment check

Confirm your uploads describe the same scope.

Before uploading SAP exports for this assessment, confirm that every file you're about to drop in describes the same operational scope. Mismatches between files cause downstream math (consumption rates, stock-on-hand, traffic-light recommendations) to be silently wrong.

  • Plant / Location / Warehouse
    Every file (MB51, IW39, Inventory Master, Material→Vendor) is filtered to the same set of plant codes. Total-inventory + plant-only-consumption is the classic mismatch.
  • Date range / period coverage
    MB51 movement window and IW39 work-order window cover the same period. Mismatched windows distort rates and WO counts.
  • Material universe
    Inventory Master is a superset of every material that appears in MB51. Missing master records cause silent drops at parse time.
  • Fleet / Equipment scope
    Fleet Master lists every unit that appears in IW39 for this assessment. Orphan units break the fleet-scope filter downstream.
